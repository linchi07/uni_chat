# Setup Your First Agent

This article will guide you in creating your very own Agent.

## Overview & Creation

Open UniChat's Agent panel and click "Create a New Agent" to enter creation mode.

First, give your Agent a name; you can also add a remark for the Agent (remarks are not sent to the AI; they are merely for your own convenience).
Click the avatar area and upload a photo to give the Agent a face, or you can simply drag and drop the avatar image here to add it.

After completing the basic creation, you need to infuse your Agent with a "soul" and capabilities. Please refer to the following modules to complete the detailed configuration:

- ⚙️ **[Model & Parameter Settings](./model-settings.md)**: This is the Agent's "brain". Choose a suitable model (like DeepSeek or GPT-4) for it, and configure various basic data parameters.
- 🗣️ **[Writing System Prompts](./system-prompt.md)**: This is the Agent's "personality". Define who it is, what its responsibilities are, and how it communicates with you via prompts.
- 👤 **[Binding User Guises](./user-identity.md)**: This is how the Agent perceives you. You can assign it a specific Persona or have it refer to you in a special "way".
