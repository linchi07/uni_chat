# User Identity Settings

This feature works in tandem with the Persona system to provide the AI with more accurate information about you as the user.

### Default Persona When Using an Agent

This project allows you to specify a default Persona. When you use this specific Agent, it will automatically switch to this Persona. (You can always change to your desired Persona using the Persona Switcher next to the chat, and changes take effect in real-time).

If left unset, it will default to using the currently active Persona to chat with the Agent.

### Persona Extra Information

Suppose you have a Persona named Little A, and when chatting with your virtual girlfriend, you want to append to your Persona: "I am your husband."
However, you don't want her calling you husband when using the exact same Little A Persona to chat with a Programmer Agent. This is where Persona Extra Information comes into play.

Persona Extra Information is not tied to the Persona itself, but to the Agent. By entering "I am your husband" in the Persona Extra Information, no matter whether you input Little A or Little B as your Persona, this specific Agent will treat you as a husband :-D.
