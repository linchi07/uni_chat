# DeepSeek
DeepSeek's official API currently requires depositing funds. Yet, considering its excellent cost-to-performance ratio, it functions as a highly formidable choice.

## Preparation
- A Chinese mainland phone number.
- A Chinese second-generation ID card for a user over 14 years old.

## Obtain an API Key

1. Visit [DeepSeek Open Platform](https://platform.deepseek.com/).
2. Create an account.
3. Once logged in, click to top-up/deposit.
4. Conclude real-name authentication and deposit funds.
5. Post-deposit, navigate into the **API Keys** menu.
6. Create an all-new API Key.

**The API Key is only revealed once. Be absolutely sure to not dismiss the modal panel before noting it aside.**

:::danger
The generated API Key must be kept strictly confidential. Do not leak it.
:::

## Configure within UNIChat

1. Enter UNIChat Settings -> API Setup -> Add Provider.
2. Pull up the predefined **DeepSeek** preset card.
3. In the API Key settings section, attach the key acquired during the previous stage.
4. Save.
