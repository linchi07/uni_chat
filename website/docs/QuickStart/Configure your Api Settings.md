# 配置API提供商

本文将会教你如何快速配置UNIChat的API提供商。

## API提供商是什么？我为什么需要？

简单来讲，UNIChat这个APP是一个“面子”，他负责处理和储存信息，提供各种高级的功能，渲染动画···


API则负责一切底层功能，AI运行在API服务商的后台，我们通过请求API来获得AI的回复。

当我们用市面上那些成品聊天机器人的时候（例如：ChatGPT，Gemini，DeepSeek，元宝····）诸如此类的APP时，他们实际上已经帮你配置好了API。这些API往往是固定的，你不能在ChatGPT中和豆包聊天对吧。
