# KnowledgeBase (本地知识库)

**KnowledgeBase** 允许你将本地的 PDF、Markdown 或 TXT 文件挂载到 Agent 上，使其具备“外挂大脑”。

## 它是如何工作的？

UNIChat 采用 **RAG (检索增强生成)** 技术。当你就某个话题提问时，系统会：
1. 在你的关联知识库中搜索最相关的片段。
2. 将这些片段作为参考上下文发送给 AI。
3. AI 根据这些真实信息给出回复。

## 关联知识库

在 UNIChat 中，你不需要频繁切换全局知识库。通过 **会话即 Agent** 的设计，你可以直接在当前会话的设置中勾选需要关联的本地文件夹。

## 最佳实践
- **分块优化**：如果文档非常大且杂乱，建议先整理成清晰的 Markdown 格式。
- **向量化模型**：确保你在设置中选择了合适的 Embedding 模型以获得最佳的搜索精度。
