# Run UNIChat on IOS or iPadOS devices

Since I do not have a paid Apple developer account, I can't publish the app to the app store. But you can still run the app on your own device.

:::tip
UNIChat has been tested on my iPhone 13 mini (IOS 26) and iPad Pro (2021) (iPadOS 17).
So it should work fine on most Apple devices. You can always try it out and report any issues to me by creating an issue on ![GitHub](https://github.com/linchi07/uni_chat)
:::
