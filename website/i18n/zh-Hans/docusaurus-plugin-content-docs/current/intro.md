# 欢迎来到 UNICHAT

**UNIChat** 是一个突破线性限制的 **Agent 级多分支对话终端**。

它不仅仅是一个聊天客户端，更是一个能够让你自由探索对话可能性的思考工具。无论你是想调优复杂的 Prompt，还是进行沉浸式的角色扮演，UNIChat 都能为你提供最顺滑的体验。
