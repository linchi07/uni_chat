# 创建你的第一个 Agent

本文将会引导你创建一个属于自己的 Agent。

## 概览与创建

打开 UniChat 的 Agent 面板，点击“创建一个新的 Agent”来进入创建模式。

首先，给 Agent 起一个名字，你也可以给 Agent 备注（备注不会被发送给 AI，只是方便记忆罢了）。
点击头像并上传一张照片来给 Agent 一个头像，你也可以直接把头像拖拽到这里来添加。

完成基础创建后，你需要为你的 Agent 注入“灵魂”与能力。请参考以下模块完成详细配置：

- ⚙️ **[配置模型与参数](./model-settings.md)**：这是 Agent 的“大脑”。为它选择一个适合的模型（如 DeepSeek 或 GPT-4），并配置各项基础参数信息。
- 🗣️ **[编写系统提示词](./system-prompt.md)**：这是 Agent 的“性格”。通过提示词定义它是谁、负责什么，以及如何与你交流。
- 👤 **[绑定用户身份](./user-identity.md)**：这是 Agent 认识你的方式。可以为其指定特定的人格，或者让它对你有特殊的“称呼”。
