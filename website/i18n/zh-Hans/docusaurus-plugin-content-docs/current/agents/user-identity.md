# 用户身份设置

该功能和 Persona 系统协同运作，共同向 AI 提供更多准确的有关你，作为用户的信息。

### 使用 Agent 时的默认人格

该项目允许你指定一个默认人格，使用该 Agent 时，会自动切换到这个人格。（你可以随时在对话旁的人格切换器上更改为你想要的人格，并且更改会实时生效）

如果不设置该项的话，会默认当前启用的人格和 Agent 对话。

### 人格额外信息

假设你有一个人格叫小 A，在和你的虚拟女友聊天的时候，你希望把人格中补充上：”我是你老公“。
但是你又不想在使用同一个小 A 人格和程序员 Agent 聊天的时候让她叫你老公。此时就可以使用人格额外信息。

人格额外信息不和 Persona 本身绑定，而是和 Agent 绑定。只要在人格额外信息中输入“我是你老公”，不论你使用的是小 A 还是小 B 作为人格输入，这个 Agent 都会把你当成老公的:-D .
