# 模型设置

Agent 需要一个模型才能运作，首先进入模型选项卡，点击添加模型来给 Agent 指定一个模型。

:::tip

根据 Agent 本身的作用，例如助手，角色扮演等，你可以通过选择不同的模型来获得较好的效果。例如 DeepSeek 在角色扮演中表现得非常优秀。

:::

通常来讲，你不需要更改其他的限制设置，在选择实际的模型之后，它们会自动按照 UNIChat 内建的模型预设自动填充。

### 模型基础信息传递

该功能可以让模型知晓一些基础信息，具体来讲是这些：
- 当前的时间和日期 （ISO 8601格式） （以便有针对性的回复，例如祝你新年快乐，或者在深夜陪你聊天……）
- 当前的操作系统和设备信息 （当你询问技术型问题时他可以针对你是 macOS 还是 Windows 用户进行不同的回复）
- 用户的语言 （防止模型用英文回复你）

一般来讲，默认开启即可。
