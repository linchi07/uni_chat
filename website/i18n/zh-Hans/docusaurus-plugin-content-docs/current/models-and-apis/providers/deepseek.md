# DeepSeek
DeepSeek官方的API需要充值，但是性价比很优秀，是不错的选择。
## 准备
- 一个中国大陆的手机号。
- 一个满14岁的中国二代身份证

## 获取 API Key

1. 访问 [DeepSeek 开放平台](https://platform.deepseek.com/)。
2. 创建一个账号。
3. 登录后，点击去充值。
4. 实名认证完成后，充值。
5. 充值之后，进入 **API Keys** 菜单。
6. 创建一个新的 API Key

**APIkey只会显示一次，切记先不要关掉页面（或者先找个地方储存好）**

:::danger
创建的 API Key，请勿泄露。
:::

## 在 UNIChat 中配置

1. 打开 UNIChat 设置 -> API设置 -> 添加提供商。
2. 找到预设的 **深度求索** 卡片
3. 在 APIkey设置 中 添加上一步获取的API Key。
4. 保存。
