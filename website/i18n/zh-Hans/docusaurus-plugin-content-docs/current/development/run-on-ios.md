# 在苹果设备（IOS 或 iPadOS）上运行 UNIChat

由于本卑微的开发者没有99美刀的充值开发者账号，所以无法上架APPStore。
但是呢，我们提供ipa文件，你可以在苹果设备上自行签名安装并使用。

ipa文件链接

关于自签名，你可以在网络上搜索或者询问AI了解更多

:::warning

自签名只能维持7天，后面需要续签。当然如果你是IOS16以下的设备，可以考虑巨魔商店。
对于非APPStore安装应用，目前有多种解决方案，不论你选择什么，务必谨慎小心，小心被骗。我们不做推荐，也不保证签名之后能用。

:::

:::tip
UNIChat 在我的 iPhone 13 mini (IOS 26) 和 iPad Pro (2021) (iPadOS 17) 上均经过了测试并且（我自己也在用）。
理论上应该能在IOS设备上良好的运行，如果你遇到任何问题或者疑问，欢迎到GitHub上提issue或者直接去QQ群问我。
