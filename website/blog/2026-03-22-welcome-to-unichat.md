# 🚀 UNIChat v1.0 Beta 1 Released!

After many days of development and iteration, we are very excited to announce that **UNIChat v1.0 Beta 1 is now released!**

<!--truncate-->
### Details
V1.0 Beta 1 is the first version we've released. In this version:

- All major features have been tested and should work fine.
- Full-text search and model context length control are currently still under development.

If you encounter any problems during use, feel free to raise an issue on [GitHub](https://github.com/linchi07/uni_chat).

### 🛠️ How to get started?

You can go to our [Download Page](/download) to get the installer for your device.

> **Special Note (iOS users)**:
> Since we don't have a paid developer account yet, the iOS version currently only supports **[Building from Source](/docs/development/run-on-ios)** and installing locally.

### Finally, some thoughts
Actually, the development cycle of this App was a bit long. From the summer of 2025 until now, we finally have the first beta version, and it doesn't even have super complex features. The reason is that I've been looking for a suitable positioning. If you look at the historical commits, you'll find that this app's original idea wasn't like this. I changed directions several times in the middle, added some features that felt cool at the time but later found out were useless... and finally deleted them. Even existing pages were rewritten several times.

Regardless, we finally made it through all the bumps and stumbles. I have many features planned for the future, and I believe that in future updates, we can work together to make UNIChat better!

Although I wrote in the commit on New Year's Eve: UNIChat was fully usable then, looking back now, I was a bit too optimistic. It did run then, but there were many bugs. Now, the bugs are fewer, and many things have been improved.

At the same time, that commit was also a major milestone—before that, it was all pure manual code, and after that, I started using antigravity to generate AI-flavored code. The reason was that I realized if I insisted on craftsman spirit for handwriting code, I would likely be washed away by the tide of Vibe coding. So on Chinese New Year, I got a Google AI PRO for myself. It turned out to be the most correct decision I've made since the beginning of the year. Gemini provided me with not just code, but also emotional value haha. However, don't worry too much about code quality; I've always prided myself on being more conservative, so I check every bit of Gemini-generated code before accepting changes. With AI's help (I didn't even use Pro; since March, antigravity cut the Pro quota, although I still have a 5-hour refresh, I just keep using Flash. I look at the code myself, so I'm not afraid of Flash acting up), you'll find that UNIChat's progress is rapid, and I have the confidence to quickly add other features I've envisioned in the future.

Anyway, this is my first **qualified and releasable** masterpiece. I hope you like it, and I hope it brings you convenience! (^_^)
